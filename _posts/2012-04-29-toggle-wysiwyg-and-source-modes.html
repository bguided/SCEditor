Code behind it

Setup the editor:

{% highlight javascript %} $(document).ready(function() { $("#demo-toggle").sceditor({ plugins: 'bbcode', toolbar: "bold,italic,underline|source", style: "/minified/jquery.sceditor.default.min.css", emoticonsRoot: "/" }); }); {% endhighlight %}

Then, to switch between the WYSIWYG mode and source mode, simply call the toggleTextMode method on the editor:

{% highlight javascript %} $("#demo-toggle").data("sceditor").toggleTextMode(); {% endhighlight %}
